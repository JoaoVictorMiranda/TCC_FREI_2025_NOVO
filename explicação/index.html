Módulo 1: Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele descreve a estrutura de uma página web usando elementos (tags).

Estrutura básica de um documento HTML

<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Minha Primeira Página</title> </head> <body> <h1>Olá, Mundo!</h1> <p>Esta é minha primeira página HTML.</p> </body> </html>

Tags HTML básicas

  • <h1> a <h6>: Títulos e subtítulos
  • <p>: Parágrafos
  • <a>: Links
  • <img>: Imagens
  • <ul>, <ol>, <li>: Listas

Exercício Prático

Crie uma página HTML simples com:

  1. Um título principal
  2. Um parágrafo de introdução
  3. Uma lista com 3 itens
  4. Um link para o Google

Módulo 2: Estruturação de Conteúdo com HTML

Elementos de seção

HTML5 introduziu elementos semânticos que ajudam a estruturar melhor o conteúdo:

  • <header>: Cabeçalho da página ou seção
  • <nav>: Menu de navegação
  • <main>: Conteúdo principal
  • <section>: Seção de conteúdo
  • <article>: Conteúdo independente
  • <aside>: Conteúdo relacionado (barra lateral)
  • <footer>: Rodapé

Exemplo de estrutura semântica

<body> <header> <h1>Meu Site</h1> <nav> <ul> <li><a href="#">Home</a></li> <li><a href="#">Sobre</a></li> <li><a href="#">Contato</a></li> </ul> </nav> </header> <main> <article> <h2>Título do Artigo</h2> <p>Conteúdo do artigo...</p> </article> <aside> <h3>Links Relacionados</h3> <ul> <li><a href="#">Link 1</a></li> <li><a href="#">Link 2</a></li> </ul> </aside> </main> <footer> <p>© 2023 Meu Site</p> </footer> </body>

Dica: Usar elementos semânticos melhora a acessibilidade e o SEO (otimização para motores de busca).

Módulo 3: Formulários HTML

Criando formulários

Formulários são essenciais para coletar dados dos usuários. O elemento <form> é usado para criar um formulário.

Elementos de formulário comuns

  • <input>: Campo de entrada (texto, email, senha, etc.)
  • <textarea>: Área de texto multilinha
  • <select>: Lista suspensa
  • <button>: Botão
  • <label>: Rótulo para elementos de formulário

Exemplo de formulário

<form action="/enviar" method="post"> <div> <label for="nome">Nome:</label> <input type="text" id="nome" name="nome" required> </div> <div> <label for="email">Email:</label> <input type="email" id="email" name="email" required> </div> <div> <label for="mensagem">Mensagem:</label> <textarea id="mensagem" name="mensagem" rows="4"></textarea> </div> <div> <label for="assunto">Assunto:</label> <select id="assunto" name="assunto"> <option value="duvida">Dúvida</option> <option value="sugestao">Sugestão</option> <option value="reclamacao">Reclamação</option> </select> </div> <button type="submit">Enviar</button> </form>

Módulo 4: Introdução ao CSS

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML. Ele controla o layout, cores, fontes e outros aspectos visuais.

Formas de adicionar CSS

  1. CSS Inline: Usando o atributo style em elementos HTML
  2. CSS Interno: Usando a tag <style> dentro do <head>
  3. CSS Externo: Usando um arquivo .css separado (recomendado)

Sintaxe básica do CSS

seletor { propriedade: valor; outra-propriedade: valor; }

Exemplo prático

/* CSS Interno */ <style> body { font-family: Arial, sans-serif; background-color: #f0f0f0; } h1 { color: navy; text-align: center; } p { color: #333; line-height: 1.5; } </style>

Exercício Prático

Crie um arquivo CSS externo e aplique estilos a uma página HTML:

  1. Defina uma fonte diferente para o corpo da página
  2. Altere a cor de todos os títulos
  3. Adicione uma margem aos parágrafos

Módulo 5: Seletores CSS e Especificidade

Tipos de seletores

  • Seletor de elemento: p { } (seleciona todos os parágrafos)
  • Seletor de classe: .destaque { } (seleciona elementos com class="destaque")
  • Seletor de ID: #cabecalho { } (seleciona o elemento com id="cabecalho")
  • Seletor de atributo: [type="text"] { } (seleciona elementos com type="text")

Combinadores

  • Descendente: div p { } (seleciona parágrafos dentro de divs)
  • Filho direto: div > p { } (seleciona parágrafos que são filhos diretos de divs)
  • Irmão adjacente: h1 + p { } (seleciona o parágrafo imediatamente após h1)

Especificidade

A especificidade determina qual regra CSS será aplicada quando houver conflito. A ordem de importância é:

  1. Estilos inline (style="")
  2. IDs
  3. Classes, atributos e pseudo-classes
  4. Elementos e pseudo-elementos

Dica: Use classes para estilizar elementos que se repetem e IDs para elementos únicos.

Módulo 6: Box Model e Layout Básico

O Box Model

Cada elemento HTML é uma "caixa" composta por:

  • Conteúdo: O conteúdo real do elemento (texto, imagem, etc.)
  • Padding: Espaço entre o conteúdo e a borda
  • Border: A borda ao redor do padding
  • Margin: Espaço externo ao redor da borda

Propriedades do Box Model

.caixa { width: 300px; /* Largura do conteúdo */ height: 200px; /* Altura do conteúdo */ padding: 20px; /* Espaço interno */ border: 2px solid #000; /* Borda */ margin: 10px; /* Espaço externo */ /* Para incluir padding e border no cálculo da largura/altura */ box-sizing: border-box; }

Display Property

  • block: Ocupa toda a largura disponível (div, p, h1-h6)
  • inline: Ocupa apenas o espaço necessário (span, a, strong)
  • inline-block: Como inline, mas permite definir width e height
  • none: Elemento não é exibido

Exercício Prático

Crie três divs com diferentes configurações de box model:

  1. Uma com margin e padding
  2. Uma com borda arredondada
  3. Uma com display inline-block

Módulo 7: Posicionamento e Flexbox

Propriedade position

  • static: Posição normal (padrão)
  • relative: Posição relativa à sua posição normal
  • absolute: Posição relativa ao ancestral posicionado mais próximo
  • fixed: Posição relativa à viewport (fica fixo na tela)
  • sticky: Híbrido entre relative e fixed

Introdução ao Flexbox

Flexbox é um modelo de layout que facilita o alinhamento e distribuição de elementos.

Container Flex (pai)

.container { display: flex; flex-direction: row; /* ou column, row-reverse, column-reverse */ justify-content: center; /* alinhamento horizontal */ align-items: center; /* alinhamento vertical */ flex-wrap: wrap; /* quebra de linha */ }

Itens Flex (filhos)

.item { flex: 1; /* capacidade de crescer/encolher */ order: 2; /* ordem de exibição */ align-self: flex-start; /* alinhamento individual */ }

Dica: Flexbox é ideal para layouts unidimensionais (uma linha ou uma coluna).

Módulo 8: Grid Layout

Introdução ao CSS Grid

CSS Grid é um sistema de layout bidimensional que permite criar layouts complexos com linhas e colunas.

Container Grid (pai)

.container { display: grid; grid-template-columns: 1fr 2fr 1fr; /* 3 colunas com proporções */ grid-template-rows: 100px auto; /* 2 linhas */ gap: 10px; /* espaçamento entre células */ }

Posicionamento de itens

.item { grid-column: 1 / 3; /* da coluna 1 à 3 */ grid-row: 1; /* apenas na linha 1 */ /* Ou usando áreas nomeadas */ grid-area: cabecalho; } .container { grid-template-areas: "cabecalho cabecalho cabecalho" "menu conteudo sidebar" "rodape rodape rodape"; }

Grid vs Flexbox

  • Flexbox: Ideal para layouts unidimensionais (uma direção)
  • Grid: Ideal para layouts bidimensionais (linhas e colunas)

Exercício Prático

  2. Uma barra lateral à esquerda
  3. Um conteúdo principal
  4. Um rodapé que ocupa toda a largura

Módulo 9: Design Responsivo

O que é Design Responsivo?

Design responsivo é uma abordagem de web design que faz com que as páginas se adaptem a diferentes tamanhos de tela e dispositivos.

Viewport Meta Tag

Essencial para sites responsivos em dispositivos móveis:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Permitem aplicar estilos CSS condicionalmente com base nas características do dispositivo:

/* Estilos para telas menores que 768px */ @media (max-width: 768px) { .menu { flex-direction: column; } .sidebar { display: none; } } /* Estilos para telas maiores que 1200px */ @media (min-width: 1200px) { .container { max-width: 1140px; margin: 0 auto; } }

Unidades Responsivas

  • %: Relativa ao elemento pai
  • vw/vh: Relativa à largura/altura da viewport
  • rem/em: Relativa ao tamanho da fonte

Dica: Adote uma abordagem "mobile-first": comece pelo design para mobile e depois adapte para telas maiores.
